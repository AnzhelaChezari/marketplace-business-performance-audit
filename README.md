# Комплексный аудит бизнес-показателей маркетплейса 🛒
Аттестационный проект: переход от интуитивного управления и Excel-отчетов к системной продуктовой аналитике на основе данных (Data-Driven подход).

## 🛠 Технологический стек
*   **SQL:** Сбор первичных данных и формирование витрин для анализа.
*   **Python (Pandas, Matplotlib, Seaborn):** Глубокий разведочный анализ и визуализация трендов.
*   **Продуктовые метрики:** Расчет Retention, Churn, LTV, CAC, ROMI.
*   **Статистика:** Проверка гипотез и оценка результатов экспериментов.

## 🎯 Бизнес-задача
Маркетплейс находится в стадии активного роста. Необходимо заменить разрозненные бухгалтерские сводки целостной системой продуктовых метрик, чтобы определить точки роста и причины оттока пользователей.
